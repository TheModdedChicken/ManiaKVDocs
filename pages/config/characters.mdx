# Characters

### Characters are bundles of textures for use on stages. To see how to add them to a config, please go [here](/config/properties#characters-object---required).

Here is an example of what a character config could look like:
```json
{
  "id": "bongoCat4k",
  "keys": 4,
  "textures": {
    "body": "bongoCat4k/body.png",
    "instrument": "bongoCat4k/instrument.png",
    "leftIdle": "bongoCat4k/leftPaw_idle.png",
    "rightIdle": "bongoCat4k/rightPaw_idle.png",
    "key1": "bongoCat4k/leftPaw_key1.png",
    "key2": "bongoCat4k/leftPaw_key2.png",
    "key1-2": "bongoCat4k/leftPaw_key1-2.png",
    "key3": "bongoCat4k/rightPaw_key3.png",
    "key4": "bongoCat4k/rightPaw_key4.png",
    "key3-4": "bongoCat4k/rightPaw_key3-4.png"
  }
}
```

## Textures

- ### `body` - Required
    - Body image
    - Example: [`bongoCat4k/body.png`](https://github.com/TheModdedChicken/ManiaKV/blob/main/examples/userdata_example/bongoCat4k/body.png)

- ### `instrument` - Required
     - Instrument image
     - Example: [`bongoCat/instrument.png`](https://github.com/TheModdedChicken/ManiaKV/blob/main/examples/userdata_example/bongoCat4k/instrument.png)

- ### `leftIdle` - Required
     - Left hand idle image
     - Exmaple: [`bongoCat/leftIdle.png`](https://github.com/TheModdedChicken/ManiaKV/blob/main/examples/userdata_example/bongoCat4k/leftPaw_idle.png)

- ### `rightIdle` - Required
     - Right hand idle image
     - Example: [`bongoCat/rightIdle.png`](https://github.com/TheModdedChicken/ManiaKV/blob/main/examples/userdata_example/bongoCat4k/rightPaw_idle.png)

- ### `key1 - key4` - Conditional
     - Paths to key images
     - Number of key textures required depends on how many keys a character has been assigned

- ### `key1-2 & key3-4` - Conditional
     - Paths to key pair images
     - Not needed by a character if only two keys were assigned to it
     - Example: 
     ```json
     {
        "id": "2k",
        "keys": 2,
        "textures": {
          "body": "bongoCat2k/body.png",
          "instrument": "bongoCat2k/instrument.png",
          "leftIdle": "bongoCat2k/leftPaw_idle.png",
          "rightIdle": "bongoCat2k/rightPaw_idle.png",
          "key1": "bongoCat2k/leftPaw_key1.png",
          "key2": "bongoCat2k/rightPaw_key2.png"
        }
     }
     ```
