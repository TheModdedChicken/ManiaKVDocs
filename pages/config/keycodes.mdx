# Keycodes

These keycodes use the en-US layout. I will slowly be rolling out different layouts which can be found in [this](https://github.com/TheModdedChicken/ManiaKV/tree/main/ManiaKV/languages) folder.

If you're looking to change the language ManiaKV uses by default then you can go [here](/config/languages)

- `BACKSPACE` - `0x08`
- `TAB` - `0x09`
- `SHIFT` - `0x10`
- `CTRL` - `0x11`
- `ENTER` - `0x0D`
- `CAPSLOCK` - `0x14`
- `SPACE` - `0x20`
