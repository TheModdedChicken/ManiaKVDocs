# Languages

## Notes

- Implementation: [`v0.1.1-lynx`](https://github.com/TheModdedChicken/ManiaKV/releases/tag/v0.1.1-lynx)


***


Languages are defined in `ManiaKV/languages` as `.json` files.
Take a look at [this](https://github.com/TheModdedChicken/ManiaKV/blob/main/ManiaKV/languages/en_us.json) language file or the [keycodes](/config/keycodes) page of the wiki for reference.

To change the language you're currently using, head to your config file (ManiaKV/config.json).
Then add the "language" property to the file and add the name of the language file you want to use:
```json
"language": "en-us"
```

Usage in typical config file:
```json
{
  "$schema": "https://raw.githubusercontent.com/TheModdedChicken/ManiaKV/main/ManiaKV/config-schema.json",
  "language": "en_us",
  "window" : {
    "width": 594,
    "height": 334
  },
  "stages": [
    {
      "id": "4k",
      "keys": [ "A", "S", "SEMI_COLON", "SINGLE_QUOTE" ],
      "characters": [ "bongoCat4k" ],
      "shortcut": [ "CTRL", "SHIFT", "4" ],
      "background": "background.png",
      "table": "table.png"
    }
  ],
  "characters": [
    {
      "id": "bongoCat4k",
      "keys": 4,
      "textures": {
        "body": "bongoCat4k/body.png",
        "instrument": "bongoCat4k/instrument.png",
        "leftIdle": "bongoCat4k/leftPaw_idle.png",
        "rightIdle": "bongoCat4k/rightPaw_idle.png",
        "key1": "bongoCat4k/leftPaw_key1.png",
        "key2": "bongoCat4k/leftPaw_key2.png",
        "key1-2": "bongoCat4k/leftPaw_key1-2.png",
        "key3": "bongoCat4k/rightPaw_key3.png",
        "key4": "bongoCat4k/rightPaw_key4.png",
        "key3-4": "bongoCat4k/rightPaw_key3-4.png"
      }
    }
  ]
}
```

## Internal Languages
To use an internal language add an asterisk to the start of the language name:
```json
"language": "*en-us"
```
